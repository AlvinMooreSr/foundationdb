    Downloads

    Client & Server Packages

    macOS

    The macOS installation package is supported on macOS 10.7+. It includes the client and (optionally) the server.

    • FoundationDB-6.2.15.pkg

    Ubuntu

    The Ubuntu packages are supported on 64-bit Ubuntu 12.04+, but beware of the Linux kernel bug in Ubuntu 12.x.

    • foundationdb-clients-6.2.15-1_amd64.deb
    • foundationdb-server-6.2.15-1_amd64.deb (depends on the clients package)

    RHEL/CentOS EL6

    The RHEL/CentOS EL6 packages are supported on 64-bit RHEL/CentOS 6.x.

    • foundationdb-clients-6.2.15-1.el6.x86_64.rpm
    • foundationdb-server-6.2.15-1.el6.x86_64.rpm (depends on the clients package)

    RHEL/CentOS EL7

    The RHEL/CentOS EL7 packages are supported on 64-bit RHEL/CentOS 7.x.

    • foundationdb-clients-6.2.15-1.el7.x86_64.rpm
    • foundationdb-server-6.2.15-1.el7.x86_64.rpm (depends on the clients package)

    Windows

    The Windows installer is supported on 64-bit Windows XP and later. It includes the client and (optionally) the server.

    • foundationdb-6.2.15-x64.msi

    API Language Bindings

    C

    FoundationDB’s C bindings are installed with the FoundationDB client binaries. You can find more details in the C API Documentation.

    Python 2.7 - 3.5

    On macOS and Windows, the FoundationDB Python API bindings are installed as part of your FoundationDB installation.

    If you need to use the FoundationDB Python API from other Python installations or paths, download the Python package:

    • foundationdb-6.2.15.tar.gz

    Ruby 1.9.3/2.0.0+

    • fdb-6.2.15.gem

    Java 8+

    • fdb-java-6.2.15.jar
    • fdb-java-6.2.15-javadoc.jar

    Go 1.11+

    The FoundationDB Go package is available on GitHub.
